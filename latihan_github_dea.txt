perintah git :
*git clone : untuk mengcopy revo yg da di server ke laptop 
*git : untuk mencatat perubahan,buar shering data,project,dll.
*git status : untuk melihat perubahan apa aja yg udah kita tambah/buat di pc 
*git add . : untuk melihat semua file yg mau di commit
*git commit -m "menambahkan 1 file" untuk menyimpan di repo local4
*git push : untuk mengirim ke repo pusat
*git pull : untuk mengambil perubahan terbaru dari repo pusat (server)
*fork : untuk mengcopy repo dari github yg lain ke github kita sendiri
